Components

React renders content onto a webpage with the use of components. Components are easy to use, can contain other components and can be as simple or complex as you want. Splitting up your webpage’s functionality into components makes it easier to debug, maintain and add features.

To create your own component, simply import React into your javascript file and create a class that extends Component. To display content, you simply override the render() function and have it return what you wish to display, usually written in the form of JSX.
