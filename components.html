Components

React renders content onto a webpage with the use of components. Components are easy to use, can contain other components and can be as simple or complex as you want them to be. Splitting up your webpage’s functionality into components makes it easier to maintain and add new features.

To create your own component, simply import react and create a class that extends ‘Component’. To make your component display content, you simply override ‘Component’’s render function and have it return what you wish to display, usually written in the form of JSX.

To define an element’s class, the syntax looks like ‘class=”name-of-class”. But because ‘class’ is a reserved word in javascript, react has opted to use ‘className’ to define an element’s class. Using ‘{}’ you are able to pass regular javascript into JSX. You can see this happening where the button defined its onClick parameter.
