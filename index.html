What is React?

React is a javascript library for building user interfaces, most commonly single page applications. It was developed by Facebook and released in 2013. Today it is used in websites such as Netflix and Instagram and is one of the most popular javascript frameworks in existence.

What can it do?

Single page applications have become more and more common in recent years. And when you have all your information on one page, it can be very difficult to manage and structure. React uses stateful modular components to help manage content, making it easy for a website to transition from one state to another.

How does it work?

React uses a syntax called JSX (Javascript extensible markup language) which is used primarily as a visual aid when building user interfaces as it resembles HTML. This is quite powerful as it connects both the view controller elements together seamlessly.

React also uses modular structures called components. Components can be rendered, modified or removed at runtime and even remember their own state. Making it easier to integrate collective data into a single webpage.

What is this guide about?

This guide will explore what the React framework is, how to get started with using it, some of the possibilities that can be created and some of the main features that makes this framework so powerful. If your looking to build an awesome single page web app or just to increase your knowledge in the world of front end development then this is the guide for you. React is a great tool with the ability to build fast scalable and reusable UI components, and after reading this guide, I hope you will have a better understanding of how to use it.
