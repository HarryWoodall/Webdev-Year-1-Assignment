React Guide

Introduction

What is React?

React is a javascript library for building user interfaces, most commonly single page applications. It was developed by facebook and released in 2013. Today it is used in websites such as Netflix and Instagram and is one of the most popular javascript frameworks in existence.

What can it do?

Single page applications have become more and more common in recent years. And when you have all your information on one page of a website, it can be very difficult to manage and structure you site accordingly. React uses stateful modular components to help manage clutter and making it easy for a website to transition from one state to another.

How does it work?

React uses a syntax called JSX (Javascript extension) which is used primarily as a visual aid when building user interfaces as it resembles HTML. This is quite powerful as it connects both the view and the controller together seamlessly.

React also uses modular structures called components. Components can rendered, modified or removed at runtime and even remember their own state. Making it easier to integrate collective data into a single webpage.
