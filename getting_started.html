React Guide

Getting Started

There are 2 ways to add React to your website. The first way involves importing small snippets of react code into an existing project, the other is making a full blown react app from scratch. The second way is more involved, but also gives you many benefits.

Adding React to an existing project

Step 1

Add a container that you wish your react component to be placed.

<!-- Other html -->
<div id="react-component-container"></div>
<!-- Other html -->

Step 2

Add Scripts to your webpage

<!-- Other html -->
<script src="https://unpkg.com/react@16/umd/react.development.js"></script>
<script src="https://unpkg.com/react-dom@16/umd/react-dom.development.js"></script>
<script src="myComponent.js"></script>

</body>

Where myComponent.js contains the react component that you wish to implement.

Step 3

Create and render your react component by putting this inside myComponent.js.

"use strict"

class HelloWorld extends React.Component {
 render() {
  return React.createElement("h1", null, "Hello World");
 }
}

const domContainer = document.getElementById("react-component-container");
ReactDom.render(React.createElement(Hello World), domContainer);

Creating a full React application

For this method, we will be using npm to install the React packages. To create a new react app, you need to install it using npm on the command line.

nxp create-react-app my-app
cd my-app

This process could take several minutes to complete to be patient. After this is complete, your main file structure should look like this.

public
 index.html
src
 app.css
 app.js
 index.css
 index.js
package.json

These are not all the files that are listed, but these are the important ones. App.js is the main pre-defined react component. Editing this file will change what you see when you run npm start when you are in the my-app directory.

As you can see, unlike the last example, this project uses JSX. That is because when you ran npx create-react-app my-app, it installed a JSX compiler that converts your project into regular javascript that a browser can understand.
